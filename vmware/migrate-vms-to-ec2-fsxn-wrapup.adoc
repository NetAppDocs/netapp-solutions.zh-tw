== 其他可能性

同樣的方法也可以延伸、使用內部部署 VM 上的客體內儲存設備來移轉 VM 。作業系統 VMDK 可以使用 CMC 移轉、且可使用 SnapMirror 複寫客體內 iSCSI LUN 。此程序需要中斷鏡射並將 LUN 附加至新移轉的 Amazon EC2 執行個體、如下圖所示。

image::migrate-ec2-fsxn-image13.png[顯示此移轉解決方案的其他可能性的影像]
