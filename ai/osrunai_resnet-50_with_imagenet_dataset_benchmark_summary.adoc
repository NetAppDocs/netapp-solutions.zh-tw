= ResNet-50搭配ImageNet資料集基準測試摘要
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
我們使用業界標準基準測試工具TensorFlow基準測試、來驗證此系統的運作和效能。ImageNet資料集用於訓練ResNet-50、這是著名的Convolutional Neurition Network（CNN）DL影像分類模式。ResNet-50提供準確的訓練結果、並可加快處理時間、讓我們能夠滿足對儲存設備的充分需求。
