= 網路裝置故障預測使用案例摘要
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


此使用案例是以亞洲電信領域的Iguazio客戶為基礎。由於每年有10萬家企業客戶和125萬次網路中斷事件、因此必須預測並採取主動行動、避免網路故障影響客戶。此解決方案提供給他們下列效益：

* 預測性網路故障分析
* 與票務系統整合
* 採取主動行動預防網路故障因為這項Iguazio實作的結果、60%的故障都是主動預防的。
