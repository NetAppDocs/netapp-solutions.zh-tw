:allow-uri-read: 
:nofooter: 
:icons: font

[role="lead"]
下列架構圖表說明在混合雲中、企業資料庫作業的典型實作方式、適用於開發/測試及災難恢復作業。

image::Hybrid_Cloud_DB_Diagram.png[混合雲 DB 圖表]

在一般業務營運中、雲端中的同步資料庫磁碟區可複製並掛載至開發/測試資料庫執行個體、以供應用程式開發或測試之用。發生故障時、即可啟動雲端中同步的資料庫磁碟區、以進行災難恢復。
